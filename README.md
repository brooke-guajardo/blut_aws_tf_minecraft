# blut_aws_tf_minecraft

## Terraform
```bash
# env vars set NOT RECOMMMENDED
export AWS_ACCESS_KEY_ID=your_aws_access_key
export AWS_SECRET_ACCESS_KEY=your_aws_secret_access_key

terraform init
terraform apply
```
